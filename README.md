# YouTube Music Modren UI

Fork by https://github.com/naikaku1/YouTube_Music-Modren-UI

Youtube LRC Generatorを連携し、もしLrclibに歌詞データが存在しなかった場合でもAIによって自動生成することでどんな動画にでも歌詞を表示することが出来ます。

[Youtube LRC Generator
](https://v2.app.lapius7.com/lrc_generator/
)
- 拡張機能側、API側双方がかなり複雑になってしまったため動作が安定しない場合があります。
- Lrclibに歌詞データが存在しなかった場合は自動的にAIによる生成が始まりますが大体数十秒ほど掛かります。

## 既知の問題

- 自動再生やリストにて自動的に次の曲に行った際に画面表示と音楽が一致しない事があることがあります。この場合ページをリロードすると正常に戻ります。
- 自動再生やリストにて自動的に次の曲に行った際、歌詞が自動生成されたあとの表示がおかしくなることがあります。この場合もページをリロードすると正常に戻ります。
